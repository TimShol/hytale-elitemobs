# EliteMobs

EliteMobs is a Hytale mod that adds tiered elite variants of NPCs with unique stats, gear, abilities, and drops.

## Documentation

Docs live in `docs/` and are built with Mintlify.

## Quick Start

- Configure elites in `main.yml`, `mobs.yml`, `abilities.yml`, and `visuals.yml`.
- Loot rules are in `drops.yml`.
- Gear catalogs are in `catalogs.yml`.

## Local Structure

- `docs/` Mintlify documentation site
- `Server/` generated templates and assets
- `*.yml` core configuration files
