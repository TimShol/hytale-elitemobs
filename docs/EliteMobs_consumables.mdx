# Consumables

Consumable tuning is in `consumables.yml` under `Consumables.byId`.

## Fields

- `horizontalSpeedMultiplier` slows movement while consuming
- `consumeDuration` controls how long the consume takes

Example:

```yaml
Consumables:
  byId:
    food_tier1:
      horizontalSpeedMultiplier: 0.7
      consumeDuration: 1.0
    big_potions:
      horizontalSpeedMultiplier: 0.7
      consumeDuration: 1.8
```
